# qoi-viewer

QOI Image viewer using qoi default implementation and Raylib.

![example](test.png)

## How to:

Requires CMake.

```console
git clone https://github.com/FInISHED-wORK/qoi-viewer.git
cd qoi-viewer
cmake .
make
./qoiviewer <input>.qoi
```
