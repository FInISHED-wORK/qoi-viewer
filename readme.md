# qoi-viewer

QOI Image viewer using qoi default implementation and Raylib.

## How to:

Requires CMake.

```console
git clone https://github.com/mscode98/qoi-viewer.git
cd qoi-viewer
cmake .
make
./qoi-viewer <input>.qoi
```
